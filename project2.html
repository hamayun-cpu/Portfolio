This project is a Social-Media-App. In which a user if not SignedIn, can SignIn or SignUp. If User is SignedIn he can create posts. He can also Follow other users. A SignedIn user will have Timeline page where he can see all the posts posted by him and the users he follows. He will also upload a profile picture and cover picture if he wants to.
